Splunk SIEM Homelab Live Threat Detection Lab

Personal Project · 2026

Built a fully functional SOC environment from scratch no tutorials, no guided labs. Configured Splunk Enterprise on Oracle Linux, set up log forwarding from a Windows host, then attacked my own environment from Kali Linux and detected every attack in real time. Documented findings as a professional incident report.

  • Configured Splunk Universal Forwarder on Windows to ingest live Security, System, and Application Event Logs into Splunk Enterprise running on Oracle Linux VM
  • Simulated SSH brute force attack using Hydra against the Linux target generated and detected 328 failed authentication attempts via custom Splunk alert
  • Performed network reconnaissance using Nmap from Kali Linux identified open ports and running services as an attacker would
  • Built 3 custom detection rules: brute force detection (linux_secure), failed Windows logins (Event ID 4625), and new account creation (Event ID 4720)
  • Created SOC Overview dashboard with panels for log volume over time, failed login tracking, and SSH brute force attempt monitoring
  • Wrote a complete professional incident report with attack timeline, MITRE ATT&CK mapping (T1110.001, T1021.004), response actions, and remediation recommendations
